دليل المطور: النشر (Hosting) مقابل قاعدة البيانات (Backend)
========================================================

هناك خلط شائع جداً عند المبتدئين بين "أين أضع ملفات الموقع" و "أين أضع بيانات الموقع". إليك التوضيح الشامل:

أولاً: GitHub ليس مجرد مخزن ملفات (GitHub Pages)
-----------------------------------------------
عندما ترفع ملفاتك على GitHub، هناك خدمتان منفصلتان تماماً:
1. مستودع الكود (Repository): هنا يظهر الكود والملفات للمطورين (وهذا ما كنت تخشى منه).
2. GitHub Pages: هي خدمة "استضافة" (Hosting). تقوم بتحويل هذه الملفات إلى رابط موقع حقيقي (مثلاً: https://murad.github.io/MyWebSite).
   - المستخدم العادي سيبحث عن الرابط الثاني ولن يرى الأكواد.
   - المطورون فقط هم من يدخلون للمستودع لرؤية الأكواد.

ثانياً: لماذا نحتاج Supabase إذاً؟
---------------------------------
GitHub Pages هو "مضيف ساكن" (Static Host). هو يستضيف ملفات الـ HTML والـ CSS والـ JS فقط.
- لا يمكن لـ GitHub Pages حفظ "صور جديدة" يرفعها المستخدم.
- لا يمكن لـ GitHub Pages تخزين "تعليقات" أو "مقالات" بشكل ديناميكي.

**هنا يأتي دور Supabase:**
نحن نقوم برفع الموقع (الملفات) على GitHub Pages، ولكن داخل الكود، نطلب من المتصفح أن يذهب و "يجلب" الصور والمقالات من Supabase.

ثالثاً: المقارنة النهائية (الخيار الأفضل لك)
-------------------------------------------

| الميزة | GitHub Pages | Supabase |
| :--- | :--- | :--- |
| الوظيفة | استضافة ملفات الموقع (HTML/CSS/JS) | استضافة البيانات والصور (Database/Storage) |
| التكلفة | مجاني تماماً | مجاني تماماً (في البداية) ولا يطلب بطاقة |
| الخصوصية | يمكن جعل الكود "خاص" (Private) ولكن النشر يتطلب Public في الخطة المجانية | البيانات محمية بكلمات مرور وبرمجة أمنية |
| سهولة التحديث | يتطلب تعديل كود ورفع (Git Push) | يتم عبر لوحة تحكم سهلة جداً |

الخطة المقترحة لك (The Ultimate Setup):
--------------------------------------
1. ارفع ملفاتك على GitHub واستخدم GitHub Pages ليصبح لديك رابط موقع رسمي. (إذا كنت لا تريد أن يرى أحد كودك، يمكنك استخدام استضافة Vercel فهي مجانية وتدعم المستودعات الخاصة Private).
2. استخدم Supabase كـ "خزنة" تضع فيها صورك ومقالاتك.

بهذه الطريقة:
- موقعك مستضاف ومتاح للجميع كرابط.
- صورك ومقالاتك تدار من "لوحة تحكم" احترافية في سوبربيس.
- لا تدفع فلساً واحداً ولا تحتاج لبطاقة بنكية.
