خطوات الأمان المتقدمة وحذف الملفات
================================

أولاً: إخفاء بيانات تيليجرام في Supabase
---------------------------------------
بدلاً من وضع التوكن في الكود، قمت بتحديث الكود ليجلبه من قاعدة البيانات. اتبع هذه الخطوات في Supabase:

1. اذهب لـ Table Editor وانشئ جدولاً جديداً باسم: secrets
2. أضف عمودين: 
   - key (نوعه text)
   - value (نوعه text)
3. قم بإلغاء خاصية RLS لهذا الجدول مؤقتاً (من خيار RLS في إعدادات الجدول) لكي يتمكن الكود من قراءته.
4. أضف سطرين (Insert Rows):
   - الأول: key هو "TELEGRAM_BOT_TOKEN" والقيمة هي التوكن الخاص بك.
   - الثاني: key هو "TELEGRAM_CHAT_ID" والقيمة هي الـ ID الخاص بك.

الآن، حتى لو فتح أي شخص كود موقعك على GitHub، لن يرى التوكن لأنه مخزن في قاعدة البيانات!

ثانياً: حذف الصور والملفات التجريبية من GitHub
-------------------------------------------
لحذف الصور التي رفعتها بالخطأ على GitHub وتنظيف مشروعك:

1. قم بحذف الصور يدوياً من مجلد المشروع على جهازك.
2. افتح PowerShell داخل مجلد المشروع.
3. اكتب الأوامر التالية بالترتيب:

   git add .
   git commit -m "Cleanup: remove test images"
   git push

هذا سيجعل GitHub يرى أنك حذفت الملفات وسيقوم بحذفها من الموقع الرسمي فوراً.

ثالثاً: تحسين عرض بطاقات الصور
------------------------------
لقد قمت بتحديث صفحة الصور (category-photos.html) لتكون البطاقات أصغر (180px) وتترتب بشكل شبكي (Grid) متقارب وأنيق، مما يسمح بعرض عدد كبير من الصور في الشاشة الواحدة.
